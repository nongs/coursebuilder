# 코스빌더 화면 명세서

> 문서 버전: 1.0  
> 대상: 단일 페이지(SPA) 웹 앱 — 라우팅 없이 `App` 하위에 전역 레이아웃으로 구성.

---

## 1. 전체 개요

| 항목 | 내용 |
|------|------|
| **앱 성격** | 강의 구조(주차·챕터·페이지·학습요소)를 설계하는 저작 도구 |
| **테마** | 다크 톤 헤더, 좌측 패널 + 중앙 편집 영역의 2단 레이아웃 |
| **전역 오버레이** | API 로딩 시 `GlobalLoader`(전체 차단), 작업 결과는 `Toast` |

---

## 2. 화면 구성도 (정보 구조)

```
┌─────────────────────────────────────────────────────────────┐
│ Header (고정 상단)                                              │
├─────────────────────────────────────────────────────────────┤
│ WeekTabs (주차 탭 + 주차 액션)                                   │
├──────────────┬──────────────────────────────────────────────┤
│ CourseWorkspace                                              │
│ ┌──────────┐ │ ┌──────────────────────────────────────────┐ │
│ │ChapterTree│ │ │ PageEditor (선택 페이지 없을 때 빈 상태)    │ │
│ │ (좌 300px)│ │ │  - 브레드크럼 (주차 > 챕터 > 페이지)        │ │
│ │           │ │ │  - 제목/설명 입력                          │ │
│ │           │ │ │  - LearningItemList                        │ │
│ └──────────┘ │ └──────────────────────────────────────────┘ │
└──────────────┴──────────────────────────────────────────────┘
```

## 3. 영역별 화면 명세

### 3.1 헤더 (`Header`)

| UI 요소 | 설명 |
|---------|------|
| 로고 | `Coursebuilder` 텍스트 |
| 배지 | `설계 모드` |
| 저장 상태 | 초기 로드 후 표시: `저장됨` / `저장필요` / 로드 중 `…` (툴팁으로 상태 설명) |
| 구분선 | 시각적 구분 |
| 강의명 | 현재 활성 강의 제목. 없으면 `강의명 없음`. 말줄임 |
| 강의명 수정 | 버튼(✎). 강의 없을 때 비활성 |
| 새 강의 만들기 | 초기 로드 전·강의 없을 때 강제 생성 모달 분기 시 비활성/제한 |
| 저장 | `localStorage`로 스냅샷 저장, 성공 시 Toast |
| 데이터 관리 | JSON 가져오기/내보내기 모달 |

**모달 연동**

- 강의명 수정 → `CourseTitleModal` (강의명 수정 / 저장)
- 새 강의 (기존 데이터 있음) → 확인 모달 후 생성 모달
- 강의 없음 → `CourseTitleModal` (생성, 닫기 불가 옵션)
- 데이터 관리 → `DataManagementModal`

---

### 3.2 주차 영역 (`WeekTabs`)

| UI 요소 | 설명 |
|---------|------|
| 주차 탭 리스트 | 숫자 버튼(1, 2, …). 활성 탭 강조. 많을 때 가로 스크롤 + 좌우 스크롤 버튼 |
| 가로 스크롤 | 휠 세로 이동을 가로 스크롤로 변환(트랙패드 대응) |
| + 주차 추가 | 활성 강의 없으면 비활성 |
| 주차 복제 | 선택 주차 복제 |
| 순서 편집 / 순서 편집 종료 | 켜면 탭만 드래그로 순서 변경, 클릭으로 주차 전환은 비활성 |
| 주차 삭제 | 주차 1개만 남으면 비활성. 삭제 시 확인 모달 |

---

### 3.3 코스 워크스페이스 (`CourseWorkspace`)

| 구역 | 클래스/역할 | 설명 |
|------|----------------|------|
| 좌측 패널 | `cb-workspace__left` | `ChapterTree` — 선택된 주차 기준 챕터·페이지 트리 |
| 중앙 메인 | `cb-workspace__main` | `PageEditor` — 페이지 및 학습요소 편집 |

---

### 3.4 챕터 트리 (`ChapterTree`)

| UI 요소 | 설명 |
|---------|------|
| 챕터 카드 | 제목 드래그 핸들로 챕터 순서 변경. 제목 클릭 시 인라인 편집(취소/확인) |
| 챕터 추가 | 챕터 추가 버튼 |
| 챕터 삭제 | 확인 모달 후 해당 챕터 및 하위 페이지·학습요소 제거 |
| 페이지 행 | 드래그 핸들로 챕터 내 순서 변경. 제목 클릭으로 페이지 선택. 삭제 버튼 |
| 페이지 추가 | 챕터별 페이지 추가 |
| 페이지 간 이동 | 동일 챕터 내 / 다른 챕터로 드래그 앤 드롭(드롭존 구분) |

**시각**  
드래그 중 카드·행 투명도 강조, 드롭 가능 영역 하이라이트.

---

### 3.5 페이지 편집기 (`PageEditor`)

| 상태 | 화면 |
|------|------|
| 페이지 미선택 | 중앙 빈 상태 일러스트 + `선택된 페이지가 없습니다.` 문구 |

| 페이지 선택 시 | 설명 |
|----------------|------|
| 브레드크럼 | `N주차` > `챕터명` > `페이지명`. 미입력 시 회색 안내 문구. 말줄임 + `title` 툴팁 |
| 페이지 제목 | 한 줄 입력 |
| 페이지 설명 | 여러 줄 텍스트area |
| 학습요소 | `LearningItemList` |

---

### 3.6 학습요소 목록 (`LearningItemList`)

| UI 요소 | 설명 |
|---------|------|
| 헤더 | 제목 `학습요소`, `+ 학습요소 추가` |
| 빈 목록 | `학습요소가 없습니다.` |
| 카드 상단 | 타입 선택(`select`), 삭제(삭제 확인 모달) |
| 타입 | 없음(선택 유도), 텍스트, 비디오, 퀴즈, 이미지, 임베드 |
| 공통 필드 | 학습요소명, 상세 내용 |
| 비디오 | YouTube URL 입력 시 미리보기(썸네일) |
| 퀴즈 | 단답형/서술형/객관식 라디오, 유형별 정답·루브릭·보기·정답 선택 |

---

### 3.7 공통 모달·토스트

| 컴포넌트 | 용도 |
|----------|------|
| `Modal` | 제목, 본문, 푸터 버튼. `closable=false` 시 백드롭/닫기 제한 |
| `Toast` | 성공/실패/일반 메시지 하단(또는 지정 위치) 알림 |
| `GlobalLoader` | `fetch`/`save` 등 API 요청 중 전체 화면 로딩 |
| `DataManagementModal` | 내보내기 파일명 타임스탬프, 가져오기 시 검증 후 덮어쓰기 확인 모달 |
| `CourseTitleModal` | 강의명(필수), 생성 시 주차 수(숫자 1~99) |

## 4. 반응형·접근성 (현재 구현 기준)

| 항목 | 내용 |
|------|------|
| 레이아웃 | 고정 그리드 `300px + 1fr`. 모바일 전용 브레이크포인트는 없음 |
| 키보드 | 모달 `role="dialog"`, `aria-modal` 등 일부 적용 |
| 포커스 | 모달 포커스 트랩은 기본 브라우저 동작 수준(추가 개선 여지) |
